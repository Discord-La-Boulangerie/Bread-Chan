Histoire de Bread Chan

Bread Chan est un bot Discord de modération qui a été créé en 2020 par une équipe de développeurs passionnés. L'idée derrière le bot était de simplifier la gestion des serveurs Discord en fournissant des outils de modération avancés tout en restant convivial et amusant.

Librairies Python Utilisées

  • discord.py
  • asyncio
  • python-dotenv

Version de Python

Bread Chan est développé en utilisant Python 3.11.2, garantissant ainsi une compatibilité avec les dernières fonctionnalités de Discord et des performances optimales.

Configuration

Pour le moment, il est impossible de configurer Bread Chan sur votre propre serveur Discord
