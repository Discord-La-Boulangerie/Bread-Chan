Commandes de Bread Chan

Découvrez les commandes puissantes de Bread Chan, le bot de modération pour plusieurs serveurs Discord. Que vous ayez besoin de gérer les membres, de personnaliser les interactions ou d'obtenir des informations, notre bot est là pour vous simplifier la tâche. Explorez les commandes disponibles pour améliorer la gestion de votre serveur Discord.

Commande Description Exemple Réponse possible
/help Affiche la liste des commandes disponibles.
/info Donne des informations générales sur le bot
/fortnite_profil Obtenir des informations sur un compte Fortnite spécifié
/genshin_profil Obtenir des informations sur un compte Genshin Impact spécifié
/bs_profil Obtenir des informations sur un compte Brawl Stars spécifié
/blague-random Donne une blague au hasard
